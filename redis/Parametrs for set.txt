ex устанавливает флаг истечения срока действия для имени ключа на ex секунд.
px устанавливает флаг истечения срока действия для имени ключа на px миллисекунд.
nx, если установлено значение True, установите значение имени ключа только для значения
если его не существует.
xx, если установлено значение True, установите значение имени ключа только для значения.
если он уже существует.
Keepttl, если True, сохраняет время жизни, связанное с ключом.
(Доступно начиная с Redis 6.0)
получить, если True, установить значение имени ключа в значение и вернуть
старое значение, сохраненное в ключе, или «Нет», если ключ не существовал. (Доступно начиная с Redis 6.2)
exat устанавливает флаг истечения срока действия для имени ключа на несколько секунд,
указано в unix-времени.
pxat устанавливает флаг истечения срока действия для имени ключа на миллисекунды,
указано в unix-времени.
